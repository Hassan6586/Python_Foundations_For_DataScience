# Lab 8: Exception Handling and Logging

## Objectives
- Implement proper exception handling using try/except blocks.
- Handle specific exceptions like `FileNotFoundError` and `JSONDecodeError`.
- Configure Python's logging module for tracking runtime events.

## Files Included
- **file_handler.py**: Basic file reading with exception handling.
- **enhanced_file_handler.py**: Uses context managers for safer file operations.
- **json_handler.py**: Basic JSON parsing validation.
- **advanced_json_handler.py**: Custom exceptions (`JSONFileError`) and class-based processing.
- **basic_logging.py**: Simple logging configuration using `basicConfig`.
- **advanced_logging.py**: Complex logging with multiple handlers (File, Stream, Rotating).

## How to Run
Run the scripts in the following order:
1. \`python3 file_handler.py\`
2. \`python3 enhanced_file_handler.py\`
3. \`python3 json_handler.py\`
4. \`python3 advanced_json_handler.py\`
5. \`python3 basic_logging.py\`
6. \`python3 advanced_logging.py\`
