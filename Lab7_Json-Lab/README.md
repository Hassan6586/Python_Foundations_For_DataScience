# Lab 7: Working with JSON Files

## Objectives
- Understand what JSON is and why it's important in data exchange
- Read JSON files and display their contents in a formatted manner
- Convert Python dictionaries to JSON strings and save them to files
- Parse and navigate through nested JSON structures
- Handle common JSON operations using Python's built-in `json` module

## Files Included
- **student_data.json**: Sample student data in JSON format.
- **read_json.py**: Script to read and display formatted JSON.
- **dict_to_json.py**: Script to convert Python dicts to JSON files.
- **company_data.json**: Complex, nested JSON data representing a company.
- **parse_nested_json.py**: Advanced script to traverse nested data.
- **json_explorer.py**: Interactive CLI tool to explore JSON structure.

## How to Run
Run the scripts in the following order:
1. \`python3 read_json.py\`
2. \`python3 dict_to_json.py\`
3. \`python3 parse_nested_json.py\`
4. \`python3 json_explorer.py company_data.json\`
